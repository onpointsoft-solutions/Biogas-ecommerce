# Biogas Technician Service App

A Flutter App For Biogas Technician Service Booking and Management üöÄ

The biogas technician app is a comprehensive service booking platform for biogas system installation, maintenance, and repair services.

This project is based on the [Flutter Getx Template](https://github.com/EmadBeltaje/flutter_getx_template) made by [Emad Beltaje](https://github.com/EmadBeltaje).

## Features üåü

- **Service Categories**: Browse biogas service categories (Repair, Setup, Purchase, Maintenance)
- **Service Booking**: View and book biogas technician services
- **Google Authentication**: Secure sign-in with Google account
- **Service Details**: View detailed information about each service including duration and pricing
- **Professional Technicians**: Connect with certified biogas technicians
- **Emergency Services**: 24/7 emergency repair services
- **Responsive Design**: Optimized for all screen sizes
- **Dark/Light Theme**: Toggle between light and dark themes

## Services Offered üîß

- **Emergency System Repair**: 24/7 repair services for biogas system malfunctions
- **Complete System Installation**: Professional installation of biogas digestion systems
- **Annual Maintenance Check**: Comprehensive maintenance and performance optimization
- **Biogas Equipment Purchase**: High-efficiency biogas digesters and components
- **Gas Line Installation**: Professional gas line installation and safety inspection
- **System Consultation**: Expert consultation for biogas system planning

## Setup Instructions ‚öôÔ∏è

1. **Clone the repository**:
   ```bash
   git clone https://github.com/onpointsoft-solutions/Biogas-ecommerce.git
   cd Biogas-ecommerce
   ```

2. **Install dependencies**:
   ```bash
   flutter pub get
   ```

3. **Firebase Setup**:
   - Create a new Firebase project at [Firebase Console](https://console.firebase.google.com/)
   - Enable Google Sign-In Authentication
   - Download the configuration files and update `firebase_options.dart`
   - Add SHA-1 and SHA-256 fingerprints for Android

4. **Run the app**:
   ```bash
   flutter run
   ```
